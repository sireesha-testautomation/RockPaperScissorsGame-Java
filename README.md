
## Rock Paper Scissors ‚Äì Java Console Game 

A simple **Rock Paper Scissors** game implemented in **Core Java** using the console.
The user plays against the computer, which makes a random choice each round.

##  Features

* Console-based interactive game
* User vs Computer gameplay
* Random computer choice using `Random`
* Input handling using `Scanner`
* Option to continue or exit after each round
* Handles invalid user input

##  Technologies Used

* Java (Core Java)
* `java.util.Scanner`
* `java.util.Random`

##  How to Play

1. Run the program.
2. Enter your choice:

   * `0` ‚Üí Rock
   * `1` ‚Üí Paper
   * `2` ‚Üí Scissors
3. The computer randomly selects its choice.
4. The winner is decided based on the rules.
5. After each round, you can choose to continue or exit.

##  Sample Output

